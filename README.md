One-page Astro mockup for a premium barber shop site in Monterrey, N.L. Includes Setmore booking, click-to-call, and WhatsApp CTAs.

## Local development

```sh
npm install
npm run dev
```

Astro runs at `http://localhost:4321` by default.

## Build

```sh
npm run build
npm run preview
```

The production build outputs to `dist/`.

## Deploy to Netlify

This repo includes `netlify.toml` with the correct settings.

1. Push the repo to your Git provider.
2. Create a new Netlify site and select the repo.
3. Build command: `npm run build`
4. Publish directory: `dist`

Netlify will use the config automatically if you keep `netlify.toml`.
